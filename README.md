# VoyageDesk — Travel Management System

A console-based travel management system built in C++ that streamlines 
core travel operations through a unified menu-driven interface.

## Features
- Tour package creation, customization, and deletion
- Booking management with auto-generated IDs
- Accommodation and transportation tracking
- Customer support ticketing system

## Tech Stack
- Language: C++
- IDE: Dev-C++ / Code::Blocks

## How to Run
1. Clone the repository
2. Open `voyagedesk.cpp` in your IDE
3. Compile and run
